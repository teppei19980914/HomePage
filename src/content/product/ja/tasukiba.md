## たすきば とは

**たすきば Knowledge Relay** は、プロジェクトの計画から実行・ふりかえりまでを一つのプラットフォームで管理しながら、そこで得た知識を組織の資産として蓄積・再利用できる業務管理ツールです。

名前の由来は「たすき（襷）」＋「場」。駅伝のたすきのように、あるプロジェクトで得た知見を次のプロジェクトチームへ手渡していく——その「つなぎの場」を提供します。

## 解決したい課題

多くの組織では、プロジェクトが終わるたびに知識が散逸します。

- 見積の精度が毎回リセットされる
- 過去に踏んだ地雷を別チームが再び踏む
- ふりかえりの結果が議事録に埋もれて活用されない

たすきばは、プロジェクト運用そのものにナレッジ循環の仕組みを埋め込むことで、**「回すほど次がうまくいく」** 状態を目指します。

## 主な機能

| カテゴリ | 機能 |
|---|---|
| **プロジェクト管理** | ステータスフロー管理・ダッシュボード |
| **見積管理** | 過去実績からの知識参照つき見積作成 |
| **WBS / タスク管理** | ガントチャート・個人タスクボード |
| **リスク・課題管理** | 早期検知と QCD バランス維持 |
| **ナレッジ管理** | 検索可能・アクセス制御つきの知識ベース |
| **ふりかえり** | プロジェクト終了時の学びを構造化して蓄積 |

## 技術スタック

- **フロントエンド**: Next.js 16 / React 19 / TypeScript / Tailwind CSS / shadcn/ui
- **バックエンド**: Next.js API Routes / Prisma 7
- **データベース**: PostgreSQL 16（pg_trgm による全文検索）
- **認証**: NextAuth.js 5（MFA / RBAC 対応）
