## プロジェクト概要

熊本県の地方創生プロジェクトとして、JR九州・チャリチャリ（シェアサイクル）・行政・金融機関が参加するコンソーシアム型の大規模案件。

自転車利用に応じてライドチケット（クーポン）を配布し、**行動変容・ウェルビーイングを促進する LINE ミニアプリ**を開発。交通渋滞解消、地域活性化、ウェルビーイングなど公共的テーマを扱う社会性の高いプロジェクト。

フェーズ型で拡張予定：5月にプロトタイプ → 10月にランキング・GPS拡張 → 将来的にバス連携・画像解析など段階的に発展。

## 担当範囲

- **技術選定・アーキテクチャ設計** — AWS EC2、Vercel、LINE LIFF SDK 統合など一人で設計
- **フロントエンド開発** — React / Vite / TypeScript
- **バックエンド開発** — Express / TypeScript / SQLite
- **インフラ構築** — EC2 セットアップ、Nginx リバースプロキシ、Docker、pm2 による本番運用環境の構築
- **ドキュメント作成** — API 仕様書、技術選定資料、デプロイ手順書、進捗管理表など多数の成果物を作成
