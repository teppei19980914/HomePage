## 連載第 8 回です

[前回](/HomePage/ja/blog/20260504-claude-prompt-thinking-patterns/)で思考系 13 パターンをお伝えしました。今回は、**技術以外の日常タスク** で Claude を使うパターン集です。

私は開発者ですが、Claude を使うのはコードだけではありません。**メール作成、キャリア相談、日記のリフレクション、旅行計画** まで、日常のあらゆるシーンで相棒として使っています。

## D. 文章作成系

### パターン 1. ブログ記事のドラフト

```
{{テーマ}} について、ブログ記事のドラフトを書いてください。

- 読者: {{想定読者}}
- トーン: {{エッセイ調 / 技術解説調}}
- 構成: 導入 → 問題提起 → 解決 → まとめ
- 読了時間目標: {{X}} 分
```

### パターン 2. メール文案

```
{{相手}} への {{用件}} のメールを書いてください。

- 関係性: {{上司 / 顧客 / 社外パートナー}}
- 目的: {{依頼 / お礼 / 謝罪 / お断り}}
- 文字数: 200 字程度
- トーン: 丁寧だが堅すぎない
```

### パターン 3. SNS 投稿文

```
{{話題}} について X で投稿する文を 3 パターン書いてください。

- 140 字以内
- ハッシュタグを 2 個
- トーン: {{カジュアル / プロフェッショナル}}
```

### パターン 4. プレゼン資料骨子

```
{{テーマ}} で 5 分 LT をします。スライド構成案をください。

- 10 枚以内
- 各スライド: タイトル + 話す内容 3 行
- 最後に CTA（行動喚起）を入れる
```

### パターン 5. 文章添削

```
次の文章を添削してください:
{{文章}}

観点:
- 冗長な表現の削除
- 誤字脱字
- 論理の飛躍
- より伝わる言い回し

出力形式: 変更箇所を diff 風に提示、理由を付記
```

## E. キャリア・人生相談系

### パターン 6. キャリアの棚卸し

```
私のキャリアを棚卸ししたいです。

現状:
- 職種: {{エンジニア}}
- 経験年数: {{X 年}}
- 得意分野: {{分野}}
- 悩み: {{言語化できる範囲で}}

質問してください:
- 私の強みを明確化するために、知りたいことを最大 5 つ挙げてください
- 質問に回答したら、強みと次に伸ばすべき方向を提示してください
```

### パターン 7. 転職検討

```
転職を検討中です。以下の条件で議論したい:

- 現職: {{業界・職種・年収・やりがい}}
- 転職先候補: {{条件}}

フレームで整理してください:
- 今の会社に残るメリット・デメリット
- 転職するメリット・デメリット
- 5 年後の自分から見たらどう評価されるか
```

### パターン 8. 副業の検討

```
副業として {{案件}} を受けるか迷っています。

- 稼働: {{時間}}
- 報酬: {{金額}}
- スキル面での学び: {{予想}}
- 本業との衝突リスク

次の軸で評価してください:
- 金銭的リターン
- スキル成長
- 時間対効果
- 健康・家族への影響
```

### パターン 9. 目標設定

```
{{期間}} の目標を立てたい。

- 仕事・健康・関係性・学習 の 4 領域で
- 各領域で SMART な目標 1 個
- 達成判定の具体基準
- 週次レビューのチェックリスト
```

### パターン 10. フィードバックの解釈

```
上司から次のフィードバックをもらいました:
{{フィードバック原文}}

以下を教えてください:
- 表面的な意味
- 背後にある本音（推測）
- 私が次に取るべきアクション 3 つ
```

## F. 感情サポート系

### パターン 11. 感情の整理

```
今、{{状況}} で {{感情}} を感じています。

- この感情の正体は何か
- 私はなぜこう感じるのか
- どうすれば楽になれるか
を整理してください。ただし、安易な「大丈夫」は言わないでください。
```

**「安易な大丈夫を言わないで」** と付けるのがコツ。Claude は共感しすぎる傾向があるので、意識的に現実を見せてもらう。

### パターン 12. 自己肯定感の回復

```
最近、{{出来事}} で自信を失っています。
私の過去の実績を一緒に振り返ってください。

質問してください:
- 過去 1 年で達成したこと
- 感謝されたこと
- 自分で誇れる瞬間
を引き出す質問を順番に 5 つ。
```

### パターン 13. 対人関係の相談

```
{{相手}} との関係で悩んでいます。

- 状況: {{客観的な事実}}
- 私の感情: {{そのときの気持ち}}
- 相手の可能性のある視点

次を教えてください:
- 相手の立場に立った解釈（最低 3 通り）
- 私が取れる行動の選択肢（最低 5 つ）
- 各選択肢の長期的な結果
```

### パターン 14. 日記のリフレクション

```
今日の出来事を 3 分で書き出しました:
{{日記の内容}}

以下を教えてください:
- 今日学んだことの本質
- 改善できる行動パターン
- 明日やってみる小さな実験
```

## G. 日常タスク系

### パターン 15. 買い物リスト生成

```
{{献立/用途}} のための買い物リストを作ってください。

- 予算: {{X 円}}
- 人数: {{Y 人}}
- 既にある食材: {{手持ち}}
- カテゴリ別にまとめ（野菜 / 肉 / 調味料 等）
```

### パターン 16. 旅行計画

```
{{行き先}} に {{期間}} 行きます。

- 予算: {{金額}}
- 興味: {{自然 / 食 / 歴史 / 温泉 等}}
- 移動手段: {{公共交通 / レンタカー}}

出力:
- 日程表（時間単位）
- 現地で食べるべきもの 3 つ
- 持ち物チェックリスト
```

### パターン 17. スケジュール整理

```
次のタスクを、今週のスケジュールに組み込んでください:
{{タスクリスト}}

- 各タスクの所要時間を見積もって
- 優先度順に並べて
- 平日 3 時間 / 週末 6 時間の枠で配分
```

### パターン 18. 習慣化サポート

```
{{習慣にしたい行動}} を続けたいです。

- 現状の頻度: {{X 回/週}}
- 目標の頻度: {{Y 回/週}}
- 障壁: {{なぜ続かないか}}

提案してください:
- トリガー設計（いつやるか）
- 最小単位（1 日何分だけなら続くか）
- ご褒美設計
- 離脱を防ぐ仕組み
```

## 私の体感 — Claude は「プライベートの相棒」

Claude を使い始めて一番驚いたのは、**プライベートでも使える** ことでした。

- 疲れて頭が回らない夜に、今日の出来事を話して整理してもらう
- キャリアで迷ったとき、同僚には相談しづらい深い話を聞いてもらう
- 目標設定のチェックインを毎週やってもらう

Claude は「**判断しない相談相手**」として優秀です。人間の友人に相談すると無意識に自分を守る言い回しになりがちですが、Claude 相手だと正直に書ける。

**AI 駆動開発は、実は「AI 駆動生活」** でもあると感じています。
