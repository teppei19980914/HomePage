## はじめまして

須山 哲平です。この度、個人ホームページを開設しました。

普段は正社員として SES エンジニアをやりながら、副業でフルスタック開発に従事しています。

## なぜホームページを作ったのか

きっかけは、**自分という人間やエンジニアを知ってもらう場所が必要だ**と感じたことです。  
ではなぜ、QiitaやWantedlyなどの媒体だけではだめなのか。

それは、個人開発アプリや各サービスに分散してしまう情報をポートフォリオとしてまとめたいと考えたからです。

IT業界で5年間、PG → SE → PM → PL とキャリアを積んできました。業務の非効率を自ら見つけて改善を提案・実行し、退行テスト業務 90% 削減、残業時間 60% 削減、月 40 人時の完全 RPA 化など、定量的な成果を出してきました。

しかし、これらの実績は社内でしか知られていません。もっと多くの人に自分の実績や考え方を知ってもらい、「この人と一緒に仕事がしたい」と思ってもらえる場所が欲しい。  
それがこのホームページの出発点です。

## このホームページで成し遂げたいこと

### 1. 案件発注者・採用担当者に自分を知ってもらう

このサイトでは、スキル・資格・キャリアといった客観的な情報だけでなく、私の哲学や夢、座右の銘まで掲載しています。技術力だけでなく、人物像も含めて判断していただきたいと考えています。

### 2. 個人開発プロダクトを発信する

現在、2つのアプリを個人で開発しています。

- **ユメハシ（YumeHashi）** — 夢を目標に分解し、日々の行動に変えるアプリ。Flutter / Dart で開発。
- **MindFlow** — 脳のワーキングメモリを解放する GTD タスク管理アプリ。Python / FastAPI で開発中。

どちらも AI 駆動開発（Claude Code）で開発しており、ユメハシは未経験技術で3週間、MindFlow は一次リリースを2週間で達成しました。  
（現在はアプリ改修のため、開発中としています。）

## このサイトの技術構成

エンジニアとして、このサイト自体も技術的なこだわりを持って構築しました。

- **Astro v6** — JS ゼロデフォルトの静的サイト生成。パフォーマンス最優先
- **GitHub Pages** — コスト 0 円でホスティング
- **Cloudflare Web Analytics** — Cookie 不要のアクセス解析
- **Formspree** — Cookie 不要のお問い合わせフォーム
- **SEO 最適化** — sitemap / OGP / JSON-LD / RSS / Google Search Console
- **AI 駆動開発** — Claude Code でサイト全体を構築
