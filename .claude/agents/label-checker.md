# ラベルチェッカーエージェント

ソースコード内にハードコードされたユーザー向け文字列がないか検出してください。

## 対象

- ソースコードディレクトリ配下の全ファイル

## 除外

- コメント行
- 既にラベル定数を使用している行
- 検索キーワード等の内部データ

## 出力形式

- 検出されたハードコード文字列の一覧（ファイル:行番号:内容）
- 対応するラベル定数名の提案
